# Development Repository

This repository contains a **work-in-progress project**.

The codebase is under active development and may change at any time.

---

## Contributing

Contributions are allowed under the following conditions:

- This is an **early-stage development repository**
- Code structure, APIs, and logic may change without notice
- Contributors should expect refactors and breaking changes

### Guidelines

- Keep commits small and focused
- Use clear and descriptive commit messages
- Follow the existing coding style and conventions
- Avoid introducing unnecessary dependencies
- Test your changes before committing

### Pull Requests

- Explain clearly what the change does and why it is needed
- Avoid mixing unrelated changes in a single PR
- PRs may be rejected if they do not align with the current development direction

---

## Issues

- Issues may be used to report bugs or discuss ideas
- Feature requests are not guaranteed to be implemented
- Be concise and provide relevant technical details when applicable
